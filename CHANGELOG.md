## [1.0.0] - 2026-04-24

버전 관리를 시작하는 기준점. 이후 모든 변경은 이 문서에 누적 기록한다.

### 주요 기능
- 월간 일정표 (지란지교소프트 품질관리팀)
- Firebase 기반 실시간 동기화 및 로그인
- 일정 CRUD: 타일 클릭 편집, D+클릭 삭제, 드래그 복사(Ctrl/Alt)
- 진행률 게이지 (예정 일정 기간 대비 현재 위치)
- 일정표 HTML 복사 (Shift+클릭 단축키)
- JSON 내보내기 / 불러오기
- 인쇄 지원
- 월간 연차 현황 플로팅 패널
- 메모 모달
- 2026~2030 한국 공휴일 내장
